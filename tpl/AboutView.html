Create a new game by specifying level of difficulty and number of players.




Users take turns flipping over 2 tiles at a time in search of a match. If you find a match, you get to go again.




When all tiles have been matched, the player with the most matches wins!


Etsy Memory Game is a sample app that demonstrates building a modern web app using public API's from Etsy. This version is built with Backbone.js and Twitter Bootstrap.
